Introduction

Webify is a cloud service provider with new technologies. Any project can be uploaded with little bit of effort.

Webify backend does the heavy work.

Install our client terminal

Install Cli

Requirements

  • Windows 10 , Mac os, Linux
  • Node.js --version 12 or higher
  • Python --version 3.6 or higher
  • Go --version 7.3 or higher

Install

Mac(homebrew)

$ brew install webify-cli

Npm

$ npm install webify-cli

Yarn

$ yarn install webify-cli

Using client terminal

Your easily use client terminal to deploy your applications in our webify cloud platform.

Install Cli

Requirements

  • Node.js --version 12 or higher
    • $ npm upload $ projectname webify-cli
  • Python --version 3.6 or higher
    • $ python upload $ projectname webify-cli
  • Go --version 7.3 or higher
    • $ go upload $ projectname webify-cli
  • php
    • $ php upload $ projectname webify-cli

Deployment

  • Sync your Github repository
    • $webify git sync project
    $webify --githubname --githubpswd --projectname sync webgit**
  • Folder from computer
  • For Node.js
    • $npm build
    $webify upload $directory --projectname sync webgit**
  • For Python
    • $python build --projectname
    $webify upload $directory --projectname sync webgit**

Databases

  • Create your table on dynamodb webify using interface
  • Use our dynamodb database
    • $webify git sync project
    $webify --githubname --githubpswd --projectname sync webgit**
    $webify use dynamodb --dbname
    $webify use dynamodb --projectname $request-method $request-page
  • Create your table on mongodb webify using interface
  • Use our Mongodb database
    • $webify git sync project
    $webify --githubname --githubpswd --projectname sync webgit**
  • Create your table on mongodb webify using interface
    • $webify use mongodb --dbname
    $webify use mongodb --projectname $request-method $request-page

Managing resource

webify allocates resources for application from pool of resources. Resource's demand is calculated based on application's past usage.

No resource is wasted

Database, authentication, cyclic graphs, automation every resources are pooled when needed. Webify backend technology is over 95% efficient to detect traffic and allocate resources.
